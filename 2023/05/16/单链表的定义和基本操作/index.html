单链表（链式存储结构）

优点：不要求大片连续空间，改变容量方便

缺点：不可随机存取，要耗费一定空间存放指针

用代码定义一个单链表

typedef <数据类型> <别名>

​ 要表示一个单链表时，只需声明一个头结点L,指向单链表的第一个结点

​ LNode *L; // 强调返回的是一个结点

或 : LinkList L; // 强调这是一个单链表

//声明一个指向单链表的第一个结点的指针
